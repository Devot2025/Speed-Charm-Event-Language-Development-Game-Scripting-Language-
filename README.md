# Speed-Charm-Event-Language-Development-Game-Scripting-Language-
Speed Charm Event is a game scripting language designed for game development.

## Current Status

### Implemented

- Basic arithmetic operations (`+`, `-`, `*`, `/`, `%`)
- System calls:
  - `__write__`
  - `__read__`
  - `__open__`

### Not Implemented

- Function bytecode generation
- Function execution
- Remaining system calls

## Test

Example script:

```text
test.sce
```
Partial memory deallocation is not yet implemented.
